--- title: "Lista completa de recursos" categories: - resources tags: - resources - recursos - software ---
Redes
  • JPerf Jperf es una interfaz gráfica de usuario basada en Java que nos permitirá utilizar el popular programa iperf para medir la velocidad máxima entre dos equipos.
Torrents
  • TransmissionQT Cliente BitTorrent muy ligero.
Firewalls
  • TinyWall Firewall muy ligero y sencillo pero con mucha potencia.
Limpiadores
  • GlaryUtilities Limpiador todo en uno con muchas funcionalidades. Muy parecido a CCleaner.
Benchmark
Analizador de hardware
  • hWiNFO Analizador de hardware y software de diagnóstico. FREEWARE.
CPU
  • OCCT Benchmark de estrés para CPUs.
GPU
  • FurMark Estresador de GPUs.
Almacenamiento
  • CrystalDiskMark Utilidad para comprobar la velocidad de los discos.
Kits completos
  • Medicat USB Kit completo para resolución de problemas, herramietnas, diagnóstico, etc. Se puede montar en un USB. IMPORTANTE: Apagar firewall durante la instalación.
  • Strelec Kit completo basado en un OS de Windows PE con muchas herramientas para recuperar archivos, limpieza, instalación de OS, etc.
Servidores
NAS
  • TrueNAS Servidor NAS de código abierto.
VPN
  • WireGuard Servidor VPN.
PXE
  • iVentoy Servidor PXE para bootear imágenes ISO desde LAN.
Plugins para navegador
  • Ghostery Bloqueador de anuncios y cookies no deseadas.
  • Twitch Ad Block Plus Bloqueador de anuncios en Twitch
Otros
  • Heidoc.net Herramientas de descargas de imágenes ISO de Microsoft Windows y Office desde fuentes oficiales.
  • Ninite Herramienta web para instalar software con un único instalador.
  • MASSGrave Activador Open Source de Windows y Office.
  • NVCleaninstall Instalador de controladores de GPUs de Nvidia.
  • Windows Re-Loader Activator Activador de Windows (fuente no oficial)
  • TransWiz Herramienta para transferir datos y configuración de un ordenador a otro.
  • Deep Freeze Herramienta para "congelar" PCs y evitar cambios no deseados.
